# ViT_experiments

Experiments related to fine-tuning a visual Transformer, by taking a Transformer pretrained on a dataset of 500 image categories, and then retraining using a subset of the categories which are very similar to the categories of another dataset, and then measuring efficacy of this fine-tuning via F1 scores etc.

**Note that the `.ipynb` notebook was created on Google Colab and does not seem to load on Github; it is located here purely for file storage reasons.**
